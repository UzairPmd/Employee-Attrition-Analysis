# Internship Python Tasks - Task 5

This folder contains Python programs and notebook cells created as part of internship practice tasks.  
The focus is on cleaning and preprocessing the Titanic dataset using pandas.

- Loading Titanic-Dataset.csv
- Checking for missing values
- Handling missing Age values by replacing with mean
- Handling missing Embarked values by replacing with mode
- Dropping unnecessary columns (like Cabin with too many missing values)
- Final cleaned dataset with 891 rows and 11 columns
