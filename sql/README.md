# 🗄️ SQL

## Overview

This folder contains SQL scripts used to analyze the Employee Attrition dataset. The queries were written in **MySQL** to answer real-world HR business questions, generate meaningful insights, and demonstrate practical SQL concepts used in data analytics.

The analysis focuses on employee demographics, attrition trends, salary distribution, job roles, departments, overtime, and other workforce-related metrics.

## 📊 Business Questions Answered

The SQL analysis addresses several HR-related business questions, including:

- Total number of employees
- Employee attrition count and rate
- Active employee count
- Department-wise employee distribution
- Department-wise attrition analysis
- Job role-wise attrition
- Gender distribution
- Age distribution
- Education field analysis
- Monthly income analysis
- Overtime impact on attrition
- Business travel analysis
- Marital status distribution
- Job satisfaction analysis
- Years at company analysis

## 🎯 Objective

The objective of this analysis is to use SQL to extract meaningful business insights from HR data and answer common workforce-related questions that support informed decision-making.
