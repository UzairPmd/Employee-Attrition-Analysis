# Cleaned Dataset

## Overview

This folder contains the processed version of the Employee Attrition dataset after completing the data cleaning and preprocessing phase.

The cleaned dataset is used for all downstream analysis and visualization tasks.

## Contents

```text
employee_attrition_cleaned.csv
```

## Data Cleaning Performed

The following preprocessing steps were applied:

- Removed duplicate records
- Handled missing values
- Removed constant and unnecessary columns
- Validated categorical values
- Standardized the dataset for analysis

## Usage

The cleaned dataset is used for:

- Exploratory Data Analysis (EDA)
- Logistic Regression Modeling
- SQL Business Analysis
- Tableau Dashboard Development

**Note:** This dataset should be used for all analyses performed in this project.
