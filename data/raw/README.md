# Raw Dataset

## Overview

This folder contains the original Employee Attrition dataset used in this project.

The dataset is stored in its original form without any modifications to preserve data integrity and ensure reproducibility throughout the analysis process.

## Contents

```text
employee_attrition.csv
```

## Purpose

The raw dataset serves as the starting point for the project and is used for:

- Data Understanding
- Data Cleaning
- Data Validation
- Exploratory Data Analysis

**Note:** This dataset should not be modified directly. All preprocessing is performed on a copy of the dataset, and the cleaned version is stored in the `cleaned` folder.
