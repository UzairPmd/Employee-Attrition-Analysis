# Task 8 - Tableau Sales & Profitability Dashboard

## Project Overview

This project analyzes the Sample Superstore dataset using Tableau Public. The objective is to help management understand sales performance, profitability, shipping trends, and product performance through interactive visualizations and dashboards.

## Tools Used

* Tableau Public
* Sample Superstore Dataset
* GitHub

## Business Questions Answered

1. Which category contributes the least revenue?
2. Which region is most profitable?
3. Which shipping method contributes the most sales?
4. Which sub-categories drive the business?
5. Which sub-categories are both high sales and high profit?

## Key Findings

* Office Supplies contributed the least revenue.
* West Region was the most profitable region.
* Standard Class generated the highest sales among shipping methods.
* Top-performing sub-categories were identified through sales analysis.
* High sales and high profit sub-categories were identified using a scatter plot.
