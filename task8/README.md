# Internship Python Tasks - Task 8

This folder contains Tableau dashboards created as part of internship practice tasks.  
The focus is on visualizing sales and profit data from the Sample Superstore dataset.

- Sales by Category – compares sales across Technology, Furniture, and Office Supplies
- Top Products by Sales – highlights sub-categories such as Phones, Chairs, Storage, Tables, and more
- Region Profitability – shows profit distribution across Central, East, South, and West regions
- Sales by Ship Mode – compares sales performance across Standard Class, Second Class, First Class, and Same Day
