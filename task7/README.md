# Internship Python Tasks - Task 7

This folder contains SQL scripts created as part of internship practice tasks.  
The focus is on understanding and applying different types of joins in SQL.

- Creating database `hotel`
- Creating tables `customers` and `orders`
- Inserting sample records into both tables
- Performing INNER JOIN to match customers with their orders
- Performing LEFT JOIN to list all customers along with their orders (including those without orders)
