# Internship Python Tasks
This repository contains a collection of Python programs created as part of internship practice tasks.  
Each file demonstrates basic programming concepts such as conditionals, loops, recursion, and mathematical logic.
- check.py – Program to check for vowels
- divisible.py – Checks divisibility of numbers.
- largest.py – Finds the largest among given numbers.
- leap.py – Determines whether a year is a leap year.
- num_pyramid.py – Prints a numeric pyramid pattern.
- pyramid.py – Prints a pyramid pattern using loops.
- sum.py – Calculates the sum of numbers.
- table.py – Generates multiplication tables.
