# Internship Python Tasks - Task 9

This folder contains Excel and Tableau visualizations created as part of internship practice tasks.  
The focus is on analyzing sales and profit data from the Sample Superstore dataset.

- Top Product Sub-Categories by Sales – bar chart and table showing Phones, Chairs, Storage, Tables, and more
- Profit vs Sales Scatter Plot – compares profitability and sales performance across product categories
- Ship Mode Sales – donut chart showing percentage distribution across Standard Class, Second Class, First Class, and Same Day
- Sales by Category – bar chart comparing Technology, Furniture, and Office Supplies
- Region Profitability – bar chart showing profit distribution across Central, East, South, and West regions
